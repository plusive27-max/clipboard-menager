# 📋 Clipboard Manager

A lightweight, privacy-first clipboard history manager for Windows that lives quietly in your system tray.

![GitHub release](https://img.shields.io/github/v/release/plusive27-max/clipboard-menager)
![Platform](https://img.shields.io/badge/platform-Windows-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Features

- 🔄 **Auto-captures** everything you copy in real time
- 🔍 **Search** through your clipboard history instantly
- 📌 **Pin** important clips so they never get lost
- 🗑 **Delete** individual items or clear all history
- ⌨️ **Global hotkey** — open with `Ctrl+Shift+V` from anywhere
- 🚀 **Starts with Windows** automatically (toggleable)
- 💾 **Persists history** between sessions
- 🔒 **100% offline** — your data never leaves your machine

## 📥 Download

Head to the [Releases](https://github.com/plusive27-max/clipboard-menager/releases/tag/v1.0.0) page and download the latest `.exe` — no install required, just run it.

## 🛠 Built With

- [Electron](https://www.electronjs.org/) — cross-platform desktop apps with web tech
- Vanilla HTML, CSS, JavaScript

## 🚀 Run from Source

```bash
git clone https://github.com/plusive27-max/clipboard-menager.git
cd clipboard-manager
npm install
npm start
```

## 📦 Build

```bash
npm run build
```

Output will be in the `dist/` folder.
